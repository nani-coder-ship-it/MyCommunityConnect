# ConnectApp Mobile

Flutter app for the community platform.

## Setup
1. Install Flutter SDK
2. Run `flutter pub get`
3. Start a device/emulator
4. Run the app

By default, it targets API at `http://10.0.2.2:4000` for Android emulator.

## Build
- Debug: `flutter run`
- Release APK: `flutter build apk`
